## 五、关键字段映射（PDF数据 → 数据库字段）

### 储蓄险 savings
```
保证现金价值(10年) → specifications.guaranteed_cash_value_10y
退保总值(20年)     → specifications.total_cash_value_20y (中性情景)
IRR（需另行计算）  → irr_20y / irr_10y
红利实现率         → dividend_fulfillment_5y
非保证占比         → specifications.non_guaranteed_ratio
早期退保损失       → max_early_exit_loss_pct（第5年：已缴保费-退保总值/已缴保费）
保单贷款成数       → loan_ltv（AIA 90%，PRU 85%，Manulife 80%）
红利可解锁         → specifications.dividend_unlockable (bool)
多元货币种数       → specifications.currency_count
受保人可变更       → specifications.insured_changeable (bool)
```

### 终身寿险 whole_life
```
保证现价           → specifications.guaranteed_cash_value_10y / timeline JSON
累积保额(100岁)    → specifications.death_benefit_100y
年缴保费           → premium_annual
身故保障是否递减   → specifications.death_benefit_decreasing (bool)
终期红利锁定       → specifications.terminal_bonus_lockable (bool)
保单逆按揭         → specifications.reverse_mortgage_eligible (bool)
额外身故保障       → specifications.extra_death_benefit_desc
```

### 重疾险 critical_illness
```
重疾病种数         → covered_conditions_count
早期危疾种数       → specifications.early_stage_conditions
多重赔付次数       → specifications.multipay_times（癌症次数）
癌症赔付次数       → specifications.cancer_multipay_times
等待期(天)         → specifications.waiting_period_days（60或90天）
保障至年龄         → specifications.coverage_to_age（100或终身）
年缴保费           → premium_annual
保证回本年         → breakeven_year（非直接字段，需计算）
```

## 七、合规注意事项

1. **数据来源**: 所有数据来自专业对比表（broker制作），基准为建议书投影值（中性情景，含非保证收益）
2. **非保证警示**: 退保总值/身故赔偿含非保证红利，实际结果可能不同
3. **版本时效**: 保司随时更新产品，录入时需注明建议书制作日期和产品版本
4. **禁止字段**: 不得录入"最优"、"推荐"、"最好"等主观判断字段
5. **比较基准**: 储蓄险为USD 100k/5年缴；重疾险为USD 100万保额/20年缴；寿险为USD 100万保额/10年缴
